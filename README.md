# Num: number utilities for mathematics

Num is a command line tool for mathematics and statistics.

    $ echo "1 2 4" | num sum
    7

    $ echo "1 2 4" | num min max range
    1 4 3

    $ echo "1 2 4" | num variance skewness kurtosis
    2.33333 1.11111 5.44444
